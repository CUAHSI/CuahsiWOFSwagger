Rest Web Services for CUAHSI Services

This is a basic set of services documentation for the Earthcube web services project. Basic CUAHSI workflow applies. Search HIS Central, and retrieve series, and service information to access these data series. Go to each service, and retrieve data.
  • Rest UI for HIS Central Catalog Web Services
  • Rest UI for a CUAHSI WaterOneFlow web service. This meets the EC Web Services API specification
For CUAHSI services the steps to discover and access data are as follows:
  • HIS Central Discovery: Get listing of terms from GetSearchableConcepts
  • HIS Central Discovery: Search for services in a small bounding
  • HIS Central Discovery: Use a term to search for series in a small bounding
  • WaterOneFlow : using the series and service inforamtion, retrieve data from distributed data services
